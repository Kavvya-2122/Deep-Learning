# Real Estate Price Prediction using Neural Network (From Scratch)

## 📌 Project Overview

This project implements a **Neural Network from scratch using NumPy** to predict real estate prices based on property features.

Unlike typical implementations, this project **does NOT use TensorFlow, Keras, or PyTorch**, and instead builds the entire deep learning pipeline manually for better understanding of core concepts.

---

## 🎯 Objectives

* Understand how neural networks work internally
* Implement forward propagation and backpropagation manually
* Compare different optimization techniques
* Analyze the effect of hyperparameters

---

## 📊 Dataset

* **Real Estate Valuation Dataset**
* Features include:

  * Transaction date
  * House age
  * Distance to MRT station
  * Number of convenience stores
  * Latitude & Longitude
* Target:

  * House price per unit area

---

## ⚙️ Features Implemented

### 🔹 Neural Network from Scratch

* Fully connected neural network
* Configurable hidden layers
* Sigmoid activation function

### 🔹 Core Concepts

* Forward Propagation
* Backpropagation
* Gradient Descent

### 🔹 Loss Function

* Mean Squared Error (MSE)

### 🔹 Optimization Techniques

* Batch Gradient Descent
* Stochastic Gradient Descent (SGD)
* Mini-Batch Gradient Descent

### 🔹 Advanced Optimizers

* Momentum
* Nesterov Accelerated Gradient

### 🔹 Experiments

* Learning rate comparison
* Hidden layer size tuning
* Optimizer performance comparison
